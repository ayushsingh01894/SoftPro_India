## 🌟 About the Internship

This repository documents my **45-Day Internship at SoftPro India**, focused on cutting-edge technologies in **Generative AI (Gen AI)** and **Agentic AI**.

Throughout this internship, I explored AI fundamentals, prompt engineering, intelligent agents, automation workflows, Python programming, and real-world AI applications.

The goal was to develop practical skills in building AI-powered solutions capable of reasoning, planning, and executing tasks autonomously.

---

## 🎯 Internship Objectives

- Understand Generative AI concepts
- Learn Prompt Engineering techniques
- Work with Large Language Models (LLMs)
- Build AI-powered Python applications
- Explore Agentic AI frameworks
- Develop intelligent autonomous agents
- Automate workflows using AI tools
- Gain hands-on industry experience
